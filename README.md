Flutter App Skeleton

This folder contains a minimal `lib/main.dart` for a simple Flutter app (counter + navigation).

Quick start (recommended):
1. Install Flutter: https://flutter.dev/docs/get-started/install
2. Create a new Flutter project and replace its `lib/main.dart` with the file in this folder, or run `flutter create .` inside this folder to generate platform files.

Windows (PowerShell) commands:

```powershell
# From this folder (flutter_app)
flutter create .
flutter pub get
flutter run
```

If you copied `lib/main.dart` into an existing project, run `flutter pub get` then `flutter run`.
